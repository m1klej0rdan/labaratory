# Библиотека StringLib для работы со строками

## Описание проекта

**StringLib** - это статическая библиотека на C++ для обработки строк, предоставляющая набор полезных функций для работы с текстом. Проект включает как саму библиотеку, так и консольное приложение для демонстрации её возможностей.

### Основные функции библиотеки:

1. **Подсчет количества слов** - определение количества слов в строке
2. **Подсчет гласных и согласных** - подсчет гласных и согласных букв
3. **Регистр строки** - перевод строки в верхний/нижний регистр
4. **Реверс строки** - переворачивание строки задом наперед
5. **Удаление лишних пробелов** - удаление двойных пробелов, пробелов в начале и конце
6. **Проверка палиндрома** - определение, является ли строка палиндромом
7. **Поиск подстроки** - подсчет количества вхождений подстроки в строку

### Особенности проекта:
- Модульная структура с разделением на библиотеку и приложение
- Использование CMake для кроссплатформенной сборки
- Поддержка русского и английского языков
- Простой и интуитивно понятный интерфейс



### Минимальные требования:
- **Компилятор C++** с поддержкой стандарта C++11:
  - GCC 4.8.1 или новее
  - Clang 3.3 или новее
  - MSVC (Visual Studio 2015 или новее)
- **CMake** версии 3.10 или выше
- **Система сборки**:
  - Make (Linux/macOS)
  - Ninja (опционально)
  - Visual Studio (Windows)

### Рекомендуемые требования:
- CMake 3.15 или выше
- GCC 9.0+ / Clang 10.0+ / MSVC 2019+
- 50 МБ свободного места на диске
- 512 МБ оперативной памяти
